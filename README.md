## HOW TO RUN IN LOCAL
1. git clone git@github.com:VanTheMan/rails-work.git
2. cd rails-work
3. bundle install
4. rake db:migrate (or bundle exec rake db:migrate)
5. rake db:seed (or bundle exec rake db:seed)
6. rails seed
